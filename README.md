## 📖 Overview

MedicalPress 2.0 is a clean, responsive, and professional healthcare website designed for:

- Hospitals
- Medical Clinics
- Healthcare Centers
- Doctors
- Telemedicine Services
- Medical Startups

The project focuses on modern UI/UX, accessibility, responsive layouts, and user-friendly navigation.

- 🏥 Modern Healthcare Landing Page
